# reMarkable Attachments

> PDFs synced automatically from the reMarkable Paper Pro via the reMarkable Sync plugin. Each PDF gets a companion `.md` note — same workflow as all other attachments.

---

## Subfolders

| Subfolder | What goes here |
|---|---|
| `Work/` | Argenta site visits, equipment sketches, meeting notes, troubleshooting diagrams |
| `Fleeting/` | Handwritten idea captures — process these into vault fleeting notes, then delete the PDF |
| `Reference/` | Handwritten reference material worth keeping long-term |

---

## Sync status

PDFs appear here automatically when you sync (`Ctrl+P` → "reMarkable Sync: Sync now" or via auto-sync).

**After a new PDF appears:**
1. Move it to the correct subfolder
2. Create a companion note using the Attachment Note Template
3. For `Fleeting/` notes: retype key ideas as vault fleeting notes (`Ctrl+Shift+F`), then archive the PDF
