### An analogy for blocks vs pages

- **Gas** -> Fleeting, ephemeral thoughts go into blocks. This is equivalent to a brain dump on the page.

- **Liquid** -> Blocks that are organised in pages or the journal. Whilst there is some organisation, there is still fluidity as blocks are easily movable.

- **Solid** -> If I'm producing some sort of output which I don't want to change, I will use a page.
