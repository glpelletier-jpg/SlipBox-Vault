## How to change Default Obsidian Theme

> [!tip]

1. If you're new to Obsidian, the default theme might feel a bit daunting at first. That's why I recommend switching it up right from the start. Simply click on the gear icon located in the bottom left corner, then navigate to the "Appearance" tab. Let's customize your experience! 🎨

[image: Obsidian-Settings-Menu-Click-SS.png]

---

# Theme Recommendation

> [!tip] I recommend starting with the [Minimal Theme](https://minimal.guide/home). That's what I use with custom css to make my theme similar to [Cybertron](https://github.com/nickmilo/Cybertron) Theme.

I would also recommend installing the "**Style Settings**" Plugin and "**Minimal Theme Settings**" Plugins which allows you to customise all your CSS Styles inside the theme without writing custom CSS.

We haven't covered installing community plugins yet but we will cover this in more detail in Section 4.

Remember you can always change this later if you like. You could also choose another theme.

---

# Install and Activate the Minimal Theme

> [!info]

1. Left Click Appearances under the Options Menu

2. Click "Manage" button

[image: Obsidian-Theme-Change-SS.png]

> [!info]

1. Type in the word "Minimal"

2. Left Click, Install and Enable Theme

[image: Obsidian-Theme-Change-Minimal-SS.png]

> [!info]

1. To ensure the Minimal Theme is enabled make sure it's selected from the Drop Down

2. In the future be sure to Check for updates here (this fixes any bugs and adds new theme features)

[image: Obsidian-Theme-Change-Minimal-Activated-SS.png]
