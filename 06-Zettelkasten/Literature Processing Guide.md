# Literature Note Processing Guide

> Which literature notes to process, in which order, and what permanent notes each will produce.
> Updated June 5, 2026. Re-evaluate after each Sunday processing session.

---

## How to use this guide

1. Pick a Tier 1 note during Sunday's weekly review (Section 8 — Zettelkasten inbox)
2. Open the literature note and read through it
3. For each claim listed below, use `Ctrl+Shift+P` to create a permanent note with that title
4. Fill in the permanent note's four sections (idea, why it matters, evidence, links)
5. Add at least 2 outbound links to other permanent notes before closing
6. Mark the claim as done in this guide with `✅`

**Time estimate:** 15–20 minutes per permanent note · 2–3 notes per Sunday = 30–45 min
