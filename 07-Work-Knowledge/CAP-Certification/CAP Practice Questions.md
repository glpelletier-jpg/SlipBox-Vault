## How to use this log

1. After each study session, add questions you got wrong (or that required guessing) to the table below
2. Record the domain, your answer, the correct answer, and **why** you got it wrong
3. Before the exam, sort by domain and review all "Why wrong" notes
4. Mark a question ✅ when you're confident you'd get it right now
