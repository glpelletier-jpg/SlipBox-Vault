# Add a new user to OT Domain

# Initial

1. Create new user name with CWID Assignment Sheet

2. After receiving the request form, ensure it has:

    1. The new user's full name

    2. The area in which they will work

    3. What groups they will need to be assigned to.

# Assign to correct domain

1. Based on the area, select the domain server

    1. SAH2PCSDC1R2 (old DN Server)

        1. This is for all Production personal and will cover 90% of the requests

            1. Area Access

            2. Access Level

                1. Operator = InBatch CompOperator

                2. Operator 2 = InBatch_CompOperatorLevel2


<!-- restart numbering -->

    1. SHA-MFG_DC1

    2. SHA-MFG_DC2




# Create user and Assign Groups

Use Production Systems Access Request Form to determine setup

- New User

- Area Access Requested (Check all that Apply)

- Area Level (Check One)

# Complete Task

1. Send new user email with new user creditisals

    1. CWID name

    2. Temporary password, with instruction s that they will need to change on first logon.

    3. Sign, date, and file request form in room # ?? (CAF 2nd floor storage room)
