# Windows 11 Clean installation

#### Before you begin

- Ensure internet access.​

- Prepare a blank USB drive (8GB+). Any content on it will be deleted during the media creation process.

- Note your current Windows 11 edition under Windows specifications in [Settings > System > About](https://aka.ms/WIPAboutSystem/).Example: Windows 11 Pro​

#### Create installation media

1. Plug the USB drive into your Windows PC.​

2. [Download the Windows 11 media creation tool](https://aka.ms/Windows11MediaCreationTool).

3. Run the tool as administrator.

4. Accept license terms.​

5. Uncheck Use the recommended options for this PC to choose your language and edition.​

6. Select your USB drive and select Next (this will format the drive and place the Windows installation media on it).​

7. Once complete, select Finish.

For more details, [visit the Download Windows 11 page](https://www.microsoft.com/software-download/windows11) and scroll down to the Create Windows 11 Installation Media section.​

#### Back up your files​

Back up all content you want to keep (photos, documents, application files, etc.) from the Windows PC you will clean install. All content on the PC will be deleted.​

 

#### Clean install Windows 11​​

1. Insert the USB drive into the Windows PC you need to clean install.​

2. Restart while holding Shift to boot into the Windows Recovery Environment (WinRE): select the Start menu, select the power icon (lower right corner), then hold the Shift key and select Restart.​

3. After the device restarts, choose to boot from the USB drive option.

4. On the Windows Setup page, select your language, time and keyboard preferences.​

5. Select Install now​​.

6. On Activate Windows, select I don’t have a product key (for reinstalling the same Windows edition).​

7. On Select the operating system you want to install, choose the same edition of Windows 11 you had on this PC (noted before you started the process).​

8. Accept Microsoft Software License Terms.

9. Select Custom: Install Windows only (advanced).

10. Choose the Primary hard drive partition to install Windows (no need to format the drive or delete existing partitions; previously existing additional partitions like Recovery partition will be reused).​

11. Select Next. Windows installation begins. Don’t turn off your PC.​

Note: If during installation your PC doesn't detect a hard drive, you may be required to load a driver. Reference the PC manufacturer’s website to find the driver.​

#### After installation​ ​

1. Your Windows PC will restart and guide you through setup. Your copy of Windows will be activated automatically once you reconnect your Microsoft account.

2. To re-enroll your device in the Windows Insider Program, go to [Settings > Windows Update > Windows Insider Program](https://aka.ms/WIPOptinSettings/) and select the channel you’d like to move to. [Review the How to preview Windows page for more details](https://www.microsoft.com/preview-windows).
