# Vault Quick Reference

> Every shortcut, button, and command in one place.

---

## Keyboard shortcuts

| Action | Hotkey |
|---|---|
| Open today's daily note | `Ctrl+D` |
| Open this week's review | `Ctrl+Shift+W` |
| New fleeting note | `Ctrl+Shift+F` |
| New permanent note | `Ctrl+Shift+P` |
| Add to daily capture | `Ctrl+Shift+C` |
| Fetch this week's calendar | `Ctrl+Shift+K` |
| Fetch next week's calendar | `Ctrl+Shift+J` |
| Command palette | `Ctrl+P` |
| Navigate back | `Alt+Left` (Obsidian default) |
| Navigate forward | `Alt+Right` (Obsidian default) |

## Commander — Left Ribbon

Five persistent buttons in the ribbon strip on the far left of the app.

| Button | Action |
|---|---|
| 🏠 Home | Opens the dashboard (`00-Home/00-Home.md`) |
| 📅 Today | Opens or creates today's daily note |
| 📆 This Week | Opens this week's weekly review |
| 🔍 Search | Opens global vault search |
| ⚡ Quick Switcher | Fast file jump |

---

## Commander — Note Toolbar

Six buttons at the top of every open note.

| Button | Action |
|---|---|
| ← Back | Navigate back in history |
| → Forward | Navigate forward in history |
| 🏠 Home | Opens the dashboard |
| 📅 Today | Opens today's daily note |
| 🕸 Graph | Opens graph view |
| ⌨ Palette | Opens command palette |

## QuickAdd commands

Open with `Ctrl+P` → type the name below.

| Command | What it does |
|---|---|
| `QuickAdd: New Fleeting Note` | Creates a fleeting note in `06-Zettelkasten/Fleeting/` |
| `QuickAdd: New Permanent Note` | Creates a permanent note in `06-Zettelkasten/Permanent/` |
| `QuickAdd: New Troubleshooting Log` | Creates a log in `07-Work-Knowledge/Troubleshooting/` |
| `QuickAdd: Add to Daily Capture` | Appends a timestamped entry to today's daily note |
| `QuickAdd: Weekly Calendar` | Fetches this week's Morgen/calendar events |
| `QuickAdd: Next Week Calendar` | Fetches next week's Morgen/calendar events |
| `QuickAdd: New Quarterly Plan` | Creates a quarterly planning note in `03-Reviews/Quarterly/` |
| `QuickAdd: New Course Package` | Creates a full 5-file course folder in `05-Resources/Courses/` |
| `QuickAdd: Full Backup` | Git push + UTF-8 ZIP export to `C:\Vault\exports\` |

## Morning startup order

**Morgen → Todoist → Obsidian**

1. **Morgen** — what's time-blocked today + Todoist tasks in one view
2. **Todoist** — process inbox, confirm priorities
3. **Obsidian** (`Ctrl+D`) — open daily note, log intention, start focus log

Never reversed. Don't open Obsidian before checking what's already committed to time.
