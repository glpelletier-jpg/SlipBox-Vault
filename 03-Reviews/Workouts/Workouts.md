## Setup note

To wire the Workout Log Template to QuickAdd:
1. `Ctrl+P` → QuickAdd: Manage Macros → Add Choice → Template
2. Name it "New Workout Log"
3. Template path: `_templates/Workout Log Template.md`
4. Output path: `03-Reviews/Workouts/{{DATE:YYYY-MM-DD}} {{VALUE}}`
5. Assign a hotkey if desired (e.g., `Ctrl+Shift+W` is taken — try `Ctrl+Shift+G`)
