MIT CS Open Course Ware Sequence of Classes

Foundational Programming: 

- [6.0001 / 6.1000](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.0001+%2F+6.1000&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIFBAB&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3)**: Introduction to Computer Science and Programming** 

(or its Python equivalent) is the standard starting point, using the Python language.

- [6.0002 / 6.1010](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.0002+%2F+6.1010&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIFxAB&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3)**:** [Introduction to Computational Thinking and Data Science](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=Introduction+to+Computational+Thinking+and+Data+Science&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIFxAC&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3) **or** [Fundamentals of Programming](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=Fundamentals+of+Programming&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIFxAD&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3)**:** 

This builds on the first course, teaching data science concepts and more advanced programming fundamentals.

Core Software Engineering & Theory:

- [6.1020](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.1020&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIIhAB&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3)**: Software Construction:** Focuses on building robust, maintainable software and follows the foundational courses. 

- [6.120 / 6.121](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.120+%2F+6.121&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIJRAB&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3) **/** [6.122](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.122&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIJRAC&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3)**: Math for CS / Intro to Algorithms / Design and Analysis of Algorithms:** This sequence introduces fundamental algorithms and data structures. 

- [6.140: Computability and Complexity](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.140%3A+Computability+and+Complexity&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIJxAB&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3)**:** A theoretical course on computation and complexity, often taken after the foundational algorithm courses. 

Lower-Level and Systems: 

- [6.190](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.190&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIOBAB&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3)**: Introduction to Low-Level Programming in C and Assembly:** Teaches low-level programming, crucial for understanding how computers work.

- [6.191](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.191&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIPBAB&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3)**: Computation Structures:** Explores computer architecture and the organization of computation.

Specialization Electives:

After the core sequence, you can explore more specialized topics such as: 

- **Machine Learning:** [6.390](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.390&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIRxAB&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3) (Intro to ML) is a popular choice.

- **Operating Systems:** [6.181](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.181&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQISxAB&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3) (Operating System Engineering).

- **Databases:** [6.583](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.583&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIThAB&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3) (Database Systems Engineering).

- **Robotics:** [6.420](https://www.google.com/search?cs=1&sca_esv=426a8004247428d8&q=6.420&sa=X&ved=2ahUKEwjBjMucqv-PAxUHnokEHccsAs0QxccNegQIUBAB&mstk=AUtExfB2OSCiRVrtyR1_8u5i0t2BPu912xJb_gTbyrPZQRYjt0ehV-AKF_jZd83pRDYIfIqkCmOYJiroWqj-nu-VaQ2gTSZVeuwhys5FkKaUPkjzdfux0ErXleL5xy9cZ-mnd2WMlPr3Wjm-juw1UghN_jeGruQ3hLxpeW9xZq87jI2xIac&csui=3) (Robotics Science and Systems).
