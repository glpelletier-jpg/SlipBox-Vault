# PLC Architecture — Resources

> Curated resources for PLC Architecture self-study — ranked by ROI, not comprehensiveness.

---

## Start here (highest ROI)

1. **Rockwell Logix Designer User Manual (9324-UM001)** — the definitive reference for Studio 5000; read alongside the emulator
2. **ISA Automation Body of Knowledge** — already in vault Books; the CAP exam's conceptual backbone
3. **Rockwell Training: ControlLogix Fundamentals (CCP143)** — hands-on lab curriculum if accessible via Argenta Rockwell account

## Practice and assessment tools

| Tool | URL | What it does |
|---|---|---|
| Studio 5000 Emulator | Bundled with Studio 5000 | Simulates ControlLogix execution without hardware |
| TIA Portal PLCSIM | Bundled with TIA Portal | Simulates S7-1500 execution |
| ISA CAP Practice Exams | https://www.isa.org/certification/cap | Official practice questions by domain |
| Quizlet — PLC Programming | quizlet.com | Community flashcard decks for IEC 61131-3 |

**Practice strategy:** Build a small working program in the emulator for every module — observation beats reading for PLC topics. Log each emulator session in the Progress Tracker.
