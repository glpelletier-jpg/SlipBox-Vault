# OT Networking — Resources

## Start here

1. **NIST SP 800-82 Rev 3** — free download; the definitive ICS security architecture reference
2. **Cisco Industrial Network Design Guide** — practical Purdue-model switching design for IE3400/IE4000
3. **ODVA EtherNet/IP Specification** — free at odva.org; CIP protocol reference

| Title | Source | Notes | Free? |
